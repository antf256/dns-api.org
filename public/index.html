DNS-API.org - An online DNS lookup service.

DNS Lookup Service

This service allows you to perform DNS-lookups via simple HTTP requests, receiving your results back as JSON-encoded objects.

The most basic usage example would look like this:

$ curl http://dns-api.org/$type/$host

For example to lookup which hosts handle email for the gmail.com domain you could make the following request:

$ curl http://dns-api.org/A/gmail.com

IPv6 lookups work exactly as you would expect, as does the lookup of TXT results too:

$ curl http://dns-api.org/AAAA/dns-api.org

$ curl http://dns-api.org/TXT/steve.org.uk
