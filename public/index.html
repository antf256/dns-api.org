DNS-API.org - An online DNS lookup service.

DNS API

About:

Lookup DNS records via HTTP, and respond with JSON-encoded results.

Usage:

$ curl http://dns-api.org/$type/$host

For example:

$ curl http://dns-api.org/A/gmail.com

Or:

$ curl http://dns-api.org/mx/googlemail.com

IPv6 lookups work, as you'd expect:

$ curl http://dns-api.org/AAAA/dns-api.org
